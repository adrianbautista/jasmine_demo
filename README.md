## Getting Started

#### Prereqs
- Ruby 2.1.0
- Rails 4.0.4

1. Clone this repo
2. `bundle install`
3. `cp config/database.yml.example config/database.yml`
4. Update `database.yml` with your own database credentials
5. `rake db:create`
6. Profit.

Now you can run the Jasmine specs using:
```shell
rake jasmine
```
